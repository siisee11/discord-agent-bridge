Your coding agents, live in Discord.

Discode connects Claude Code, Codex, and OpenCode sessions to Discord so your team can monitor progress, share context, and unblock faster from anywhere.

Agent Flexibility

Use Claude Code, Codex, or OpenCode with auto-detection and a unified command flow.

How the flow works

  1. 1Run discode setup YOUR_BOT_TOKEN once.
  2. 2Move into your repo and run discode go.
  3. 3Watch session updates land in your Discord project channel.

Quick install

bun add -g discode
discode setup YOUR_DISCORD_BOT_TOKEN
cd ~/projects/your-app
discode go
